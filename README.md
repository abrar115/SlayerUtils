SlayerUtils This module, based on chattriggers is aimed to help slayers in hypixel skyblock, a few features might not work as I  have noone to test it with :( and i havent found anything wrong with it yet, theres very limited features atm as its still a work in progress, the only notable features are, specific personal bests for each slayer time, except vampire, cba to do that rn :D and the average spawn, kill and total time of the last 5 bosses [commands are /avt (last 5 bosses), /pb(r,t,s,e,b) =>{r = rev, t = tara, s = sven, e = eman, b = blaze}] thank you for coming to my TedTalk [color scheme is a work in progress]
